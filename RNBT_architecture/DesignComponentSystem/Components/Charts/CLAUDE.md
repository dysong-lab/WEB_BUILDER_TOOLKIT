# Charts

## 정의

> MD3 공식 범주가 아님
> Charts visualize data through graphical representations such as bars, lines, pies, and other formats.

## 역할

데이터를 Bar, Line, Pie, Area, Scatter, Radar 등 시각적 그래프로 표현한다.
대시보드, 분석 화면, 리포트 등에서 사용한다.

## 하위 컴포넌트

- `BarChart` — 카테고리 축 기반 비교용 막대 차트
- `LineChart` — 카테고리 축 기반 추세용 라인 차트
- `PieChart` — 항목 비중 분포용 도넛 차트
- `AreaChart` — 면적 채움 기반 용량/사용량 추세 차트
- `ScatterChart` — 2개 점군 분포 비교용 산점도
- `RadarChart` — 다축 프로파일 비교용 레이더 차트
