# STATCOM_MMC

## 컴포넌트 정보

| 항목 | 값 |
|------|-----|
| 유형 | 컨테이너 (1 GLTF = N Mesh) |
| 기본 Mixin | MeshStateMixin |
| 특징 | STATCOM MMC(Modular Multilevel Converter) 장비 내부 14개 Mesh — 본체 파트(`STATCOM_MMC_00`, `_03`, `_32`~`_40`, `_77`)와 유체 흐름 표현 Mesh(`STATCOM_MMC_1257_Flow01`, `STATCOM_MMC_1258_Flow02`). mesh name은 모델 원본 식별자(`STATCOM_MMC_{n}` 형식) 그대로 유지하여 데이터 제공자 측에서 키로 매핑 |
