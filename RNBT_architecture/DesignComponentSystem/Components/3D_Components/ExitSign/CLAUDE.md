# 컴포넌트 정보

| 항목 | 값 |
|------|-----|
| 유형 | 개별 (1 GLTF = 1 Mesh) |
| 기본 Mixin | MeshStateMixin |
| meshName | exit |

## 장비 개요

ExitSign은 비상 탈출 방향을 안내하는 비상구 표지(Exit Sign) 장비를 표현하는 3D 컴포넌트이다. 단일 GLTF 모델이 단일 Mesh(`exit`)로 구성되며, MeshStateMixin을 통해 상태에 따라 메시 색상을 변경한다.
