# Dialogs

## MD3 정의

> Dialogs provide important prompts in a user flow.

## 역할

사용자 흐름에서 중요한 프롬프트를 제공하여, 사용자가 정보에 따라 행동하도록 한다.

## 핵심 특성

- 하나의 단일 작업 완료에 전용
- 작업과 관련된 정보를 함께 표시 가능
- 진행 삭제 등 고위험 액션 확인에 주로 사용

## 하위 범주

| 하위 범주 | 역할 |
|----------|------|
| BasicDialog | 중앙 모달로 확인/취소 등 단순 프롬프트 제공 |
| FullScreenDialog | 전체 화면으로 복잡한 단일 작업 수행 (앱바 포함) |
