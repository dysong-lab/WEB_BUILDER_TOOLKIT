# BasicDialog

## MD3 정의

> Basic dialogs interrupt users with urgent information, details, or actions in a centered modal.

## 역할

화면 중앙 모달로 긴급 정보, 세부 사항, 확인/취소 등 단순 프롬프트를 제공한다.

## 핵심 특성

- 중앙 모달 + scrim 오버레이로 표시
- 선택적 아이콘, 타이틀, 본문 텍스트 구성
- 최대 2개의 액션 버튼 (확인/취소)
- 진행 삭제 등 고위험 액션의 확인에 주로 사용
- 외부 탭 또는 취소 버튼으로 닫기 가능
