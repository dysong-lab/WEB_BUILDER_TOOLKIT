# Rich

## MD3 정의

> Rich tooltips provide additional context about a feature or element, and can include actions.

## 역할

기능이나 요소에 대한 상세 정보를 제공하며, 선택적으로 버튼이나 링크를 포함할 수 있다.

## 핵심 특성

- 선택적 타이틀 + 본문 텍스트 구성
- 선택적 액션 버튼 또는 링크 포함 가능
- Plain보다 넓은 영역을 차지하며 여러 줄 텍스트 지원
- 사용자가 툴팁 위로 마우스를 이동하면 유지됨 (상호작용 가능)
