# Toolbars

## MD3 정의

> Toolbars display frequently used actions relevant to the current page.

## 역할

현재 페이지와 관련된 자주 사용하는 액션을 표시한다.

## 핵심 특성

- MD3 유형: Docked toolbar, Floating toolbar
- Vibrant color 스타일로 더 큰 강조 가능
- 버튼, 아이콘 버튼, 텍스트 필드 등 다양한 컨트롤 유형 표시 가능
- FAB와 페어링하여 특정 액션 강조 가능
- Navigation bar와 동시에 표시하지 않음
