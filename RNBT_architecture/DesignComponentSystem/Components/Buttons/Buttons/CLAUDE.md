# Buttons

## MD3 정의

> Buttons prompt most actions in a UI.

## 역할

UI에서 대부분의 액션을 촉발하는 기본 버튼이다.

## 핵심 특성

- 2가지 변형: Default, Toggle
- 선택적 선행 아이콘 포함 가능
- 5가지 색상 옵션: Elevated, Filled, Tonal, Outlined, Text
- 5가지 크기 권장: Extra small, Small, Medium, Large, Extra large
- 2가지 형태 옵션: Round, Square
- 레이블은 간결하게, 문장 대소문자(sentence case) 사용

## 세부 컴포넌트

| 컴포넌트 | 역할 |
|----------|------|
| Standard | 기본 액션 버튼을 표시하고 클릭 이벤트를 발행 |
