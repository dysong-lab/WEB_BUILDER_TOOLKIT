# IconButtons

## MD3 정의

> Icon buttons help people take minor actions with one tap.

## 역할

한 번의 탭으로 부수적 액션을 수행할 수 있게 한다.

## 핵심 특성

- 명확한 의미의 시스템 아이콘만 사용
- 2가지 변형: Default, Toggle
- 다양한 구성: Color, Size, Width, Shape
- 웹에서 호버 시 액션을 설명하는 툴팁 표시
- Toggle에서 미선택 상태는 Outlined 스타일, 선택 상태는 Filled 스타일 사용

## 세부 컴포넌트

| 컴포넌트 | 역할                                                     |
| -------- | -------------------------------------------------------- |
| Standard | 아이콘 기반 보조 액션 버튼을 표시하고 클릭 이벤트를 발행 |
