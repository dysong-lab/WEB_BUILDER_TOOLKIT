# Buttons

## MD3 정의

> Buttons help people take actions, such as sending an email, sharing a document, or liking a comment.

## 역할

사용자가 이메일 전송, 문서 공유, 좋아요 등 액션을 실행할 수 있게 한다.
액션의 중요도에 따라 버튼 유형을 선택한다.

## 하위 범주

| 컴포넌트 | 역할 |
|----------|------|
| ButtonGroups | 버튼을 그룹화하고 상호작용 추가 |
| Buttons | UI에서 대부분의 액션을 촉발하는 기본 버튼 |
| ExtendedFABs | 레이블 텍스트가 필요한 주요 액션 |
| FABMenu | FAB에서 열리는 관련 액션 메뉴 |
| FAB | 화면의 가장 일반적이거나 중요한 주요 액션 |
| IconButtons | 한 번의 탭으로 부수적 액션 수행 |
| SegmentedButtons | 옵션 선택, 뷰 전환, 요소 정렬 |
| SplitButtons | 관련 액션 메뉴를 여는 분할 버튼 |
