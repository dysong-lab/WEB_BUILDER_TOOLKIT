# SegmentedButtons

## MD3 정의

> Segmented buttons help people select options, switch views, or sort elements.

## 역할

옵션 선택, 뷰 전환, 요소 정렬을 돕는다.

## 핵심 특성

- 아이콘, 레이블 텍스트, 또는 둘 다 포함 가능
- 2가지 변형: Single-select, Multi-select
- 2~5개 항목의 단순한 선택에 사용 (더 많거나 복잡한 선택은 Chips 사용)

## 세부 컴포넌트

| 컴포넌트 | 역할                                             |
| -------- | ------------------------------------------------ |
| Standard | 세그먼트 목록을 렌더링하고 단일 선택 상태를 관리 |
