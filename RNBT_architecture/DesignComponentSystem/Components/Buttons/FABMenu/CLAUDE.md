# FABMenu

## MD3 정의

> The floating action button (FAB) menu opens from a FAB to display multiple related actions.

## 역할

FAB에서 열려 2~6개의 관련 액션을 화면 위에 떠있는 형태로 표시한다.

## 핵심 특성

- FAB에서 열려 2~6개 관련 액션을 표시
- 모든 FAB 크기에 대해 하나의 FAB menu 크기 사용
- Extended FAB와는 함께 사용하지 않음
- Primary, Secondary, Tertiary 색상 세트 사용 가능
