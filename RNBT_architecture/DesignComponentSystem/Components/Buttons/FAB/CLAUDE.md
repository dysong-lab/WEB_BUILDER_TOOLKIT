# FAB

## MD3 정의

> Floating action buttons (FABs) help people take primary actions.

## 역할

화면에서 가장 일반적이거나 중요한 주요 액션에 사용한다.

## 핵심 특성

- FAB의 아이콘은 명확하고 이해하기 쉬워야 함
- 콘텐츠 스크롤 시에도 화면에 유지
- 3가지 변형: FAB, Medium FAB, Large FAB

## 세부 컴포넌트

| 컴포넌트 | 역할                                                       |
| -------- | ---------------------------------------------------------- |
| Standard | 주요 액션용 원형 플로팅 버튼을 표시하고 클릭 이벤트를 발행 |
