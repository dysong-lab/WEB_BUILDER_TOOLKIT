# Chips

## MD3 정의

> Chips help people enter information, make selections, filter content, or trigger actions.

## 역할

특정 컨텍스트의 옵션을 표시하여 정보 입력, 선택, 필터링, 액션 트리거를 돕는다.

## 하위 범주

| 하위 범주 | 역할 |
|----------|------|
| Assist | 사용자 요청에 따른 보조 액션 트리거 |
| Filter | 콘텐츠를 조건으로 필터링하는 토글 선택 |
| Input | 사용자 입력을 태그로 표현하고 삭제 가능 |
| Suggestion | 입력 시 자동완성 제안 표시 |
