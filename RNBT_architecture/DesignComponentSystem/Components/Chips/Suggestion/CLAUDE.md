# Suggestion

## MD3 정의

> Suggestion chips help narrow a user's intent by presenting dynamically generated suggestions.

## 역할

동적으로 생성된 제안을 표시하여 사용자의 의도를 구체화하는 데 도움을 준다.

## 핵심 특성

- 사용자 입력이나 컨텍스트에 따라 동적으로 생성
- 탭 시 제안 내용을 입력 필드에 반영하고 사라짐 (토글 아님)
- 아이콘 없이 텍스트만으로 구성되는 것이 일반적
- 메시지 답장, 검색 자동완성 등에서 사용
