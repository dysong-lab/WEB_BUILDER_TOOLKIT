# Input

## MD3 정의

> Input chips represent discrete pieces of information entered by a user, such as entities or attributes in a search field.

## 역할

사용자가 입력한 정보를 이산적인 태그로 표현하며, 개별 삭제가 가능하다.

## 핵심 특성

- 사용자 입력(텍스트, 선택 등)을 태그로 변환하여 표시
- 후행 삭제 아이콘으로 개별 제거 가능
- 선택적 선행 아바타/아이콘으로 입력 유형 표시
- 검색 필드, 이메일 수신자 등의 컨텍스트에서 사용
